## CSS
: html으로는 내용을 적고, css로는 웹 문서의 디자인을 구성한다.
기본적으로 선택자, 속성, 값으로 작성한다.
```css
table{
  color: gray;
  border: 1px;
}
```
: 여기서 table이 선택자이고, color, border는 속성, gray, 1px는 값이다.
